# Projeto FlashCards - Udacity

Bem vindo ao meu projeto flashcards do programa de NanoDegree Desenvolvedor React da Udacity!

Para executar o projeto, faça o clone ou baixe o .zip

Depois dentro da pasta do repositório execute

    npm install
	npm start

Depois selecione o seu emulador ou abra o expo para escanear o QRCode exibido!

# Plataformas testadas

O App foi testado apenas dentro do simulador para iPhones.
